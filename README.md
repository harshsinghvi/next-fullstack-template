# next-fullstack-template
> Template for building Full-stack application using shardcn/ui, next-auth, next js 13

## Modules
- shardcn ui
- next-auth
- next-auth mongodbadapter
- tailwind

## env example,

```.env.local
MONGODB_URI=
MONGODB_DATABASE_NAME=

NEXTAUTH_SECRET=

GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
```
